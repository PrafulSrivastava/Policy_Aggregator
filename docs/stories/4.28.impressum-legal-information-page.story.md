# Story 4.28: Impressum / Legal Information Page

## Status
Draft

## Story
**As a** user,  
**I want** to access legal company information (Impressum),  
**so that** I can identify the service provider and comply with legal disclosure requirements.

## Acceptance Criteria

1. Impressum page accessible via /#/legal/impressum route
2. Page displays required legal information:
   - Company name / Service provider name
   - Registered address
   - Contact information (email, phone if applicable)
   - Registration number (if applicable)
   - Tax ID / VAT number (if applicable)
   - Responsible person / Managing director
3. Page uses design system styling (Minimalist Monochrome)
4. Link to impressum available in footer
5. Information formatted clearly and readably
6. Last updated date displayed
7. Print-friendly formatting
